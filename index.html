Softmax损失函数

Softmax函数

Softmax函数将神经网络的原始输出（未归一化的对数概率）转换为一个概率分布：

y^j=eoj∑k=1qeok\hat{y}_j = \frac{e^{o_j}}{\sum_{k=1}^q e^{o_k}} y^​j​=∑k=1q​eok​eoj​​

其中：

  • ojo_joj​：第j个类别的得分
  • y^j\hat{y}_jy^​j​：预测概率
  • qqq：类别总数

最大似然估计（MLE）的目标

似然函数（假设样本独立同分布）：

P(Y∣X)=∏i=1nP(y(i)∣x(i))P(Y \mid X) = \prod_{i=1}^n P(y^{(i)} \mid x^{(i)}) P(Y∣X)=i=1∏n​P(y(i)∣x(i))

对数似然函数：

log⁡P(Y∣X)=∑i=1nlog⁡P(y(i)∣x(i))\log P(Y \mid X) = \sum_{i=1}^n \log P(y^{(i)} \mid x^{(i)}) logP(Y∣X)=i=1∑n​logP(y(i)∣x(i))

Softmax损失函数（交叉熵损失）

L(y,y^)=−∑j=1qyjlog⁡y^jL(y, \hat{y}) = -\sum_{j=1}^q y_j \log \hat{y}_j L(y,y^​)=−j=1∑q​yj​logy^​j​

其中：

  • yjy_jyj​：真实标签的独热编码（正确类别为1，其余为0）
  • y^j\hat{y}_jy^​j​：Softmax输出的预测概率

数学推导与梯度

损失函数对未归一化得分ojo_joj​的梯度：

∂L∂oj=y^j−yj\frac{\partial L}{\partial o_j} = \hat{y}_j - y_j ∂oj​∂L​=y^​j​−yj​

实际应用注意事项

正则化：

  • 通常加入L2正则化项防止过拟合
  • 相当于在MLE框架下引入高斯先验（最大后验估计，MAP）

数值稳定性：

  • 实现时需对Softmax的指数计算进行数值优化
  • 常用技巧：减去最大值，避免数值溢出

核心联系总结

方面 说明
概率视角 Softmax输出概率分布，MLE最大化数据似然
优化视角 交叉熵损失最小化等价于对数似然最大化
梯度性质 梯度y^j−yj\hat{y}_j - y_jy^​j​−yj​直接反映概率差异
正则化 L2正则对应高斯先验，改善泛化能力

对数似然到交叉熵的推导

最大似然估计的设定

在softmax回归中，我们试图最大化观察到当前数据标签的条件概率。假设我们有数据集 {X,Y}\{ \mathbf{X}, \mathbf{Y} \}{X,Y}，其中包含 nnn 个样本，每个样本由特征向量 x(i)\mathbf{x}^{(i)}x(i) 和对应的**独热编码（one-hot encoding）**标签向量 y(i)\mathbf{y}^{(i)}y(i) 组成。标签向量 y(i)\mathbf{y}^{(i)}y(i) 是一个 qqq 维向量（qqq 是类别数量），其中只有真实类别对应的位置为 1，其余位置为 0。

我们的目标是找到模型参数，使得给定输入 X\mathbf{X}X 时观察到标签 Y\mathbf{Y}Y 的联合条件概率最大。假设样本是独立同分布的，这个联合概率可以表示为每个样本条件概率的连乘积：

P(Y∣X)=∏i=1nP(y(i)∣x(i))P(\mathbf{Y} \mid \mathbf{X}) = \prod_{i=1}^n P(\mathbf{y}^{(i)} \mid \mathbf{x}^{(i)}) P(Y∣X)=i=1∏n​P(y(i)∣x(i))

负对数似然与损失函数

在优化中，我们通常倾向于最小化一个损失函数，而不是最大化似然函数。由于对数函数是单调递增的，最大化似然函数等价于最大化对数似然函数，同时也等价于最小化负对数似然函数（Negative Log-Likelihood, NLL）。

因此，我们对上述联合概率取负对数：

−log⁡P(Y∣X)=−log⁡[∏i=1nP(y(i)∣x(i))]=−∑i=1nlog⁡P(y(i)∣x(i))-\log P(\mathbf{Y} \mid \mathbf{X}) = -\log \left[ \prod_{i=1}^n P(\mathbf{y}^{(i)} \mid \mathbf{x}^{(i)}) \right] = -\sum_{i=1}^n \log P(\mathbf{y}^{(i)} \mid \mathbf{x}^{(i)}) −logP(Y∣X)=−log[i=1∏n​P(y(i)∣x(i))]=−i=1∑n​logP(y(i)∣x(i))

这里的 −∑i=1nlog⁡P(y(i)∣x(i))-\sum_{i=1}^n \log P(\mathbf{y}^{(i)} \mid \mathbf{x}^{(i)})−∑i=1n​logP(y(i)∣x(i)) 就是所有样本的总负对数似然损失。对于单个样本 (i)(i)(i)，其损失为 l(y(i),y^(i))=−log⁡P(y(i)∣x(i))l(\mathbf{y}^{(i)}, \hat{\mathbf{y}}^{(i)}) = -\log P(\mathbf{y}^{(i)} \mid \mathbf{x}^{(i)})l(y(i),y^​(i))=−logP(y(i)∣x(i))。

连接似然与Softmax输出概率

在softmax回归中，模型通过softmax函数将输出（logits）转换为一个概率分布。对于样本 iii，其对于类别 jjj 的预测概率为：

y^j(i)=P(y(i)=j∣x(i))=exp⁡(oj(i))∑k=1qexp⁡(ok(i))\hat{y}_j^{(i)} = P(y^{(i)}=j \mid \mathbf{x}^{(i)}) = \frac{\exp(o_j^{(i)})}{\sum_{k=1}^q \exp(o_k^{(i)})} y^​j(i)​=P(y(i)=j∣x(i))=∑k=1q​exp(ok(i)​)exp(oj(i)​)​

其中 o(i)\mathbf{o}^{(i)}o(i) 是模型对于样本 iii 的原始输出（logits）。因此，整个预测概率向量 y^(i)\hat{\mathbf{y}}^{(i)}y^​(i) 可以看作是给定输入 x(i)\mathbf{x}^{(i)}x(i) 时标签的条件概率分布的一个模型估计，即 P(y(i)∣x(i))≈y^(i)P(\mathbf{y}^{(i)} \mid \mathbf{x}^{(i)}) \approx \hat{\mathbf{y}}^{(i)}P(y(i)∣x(i))≈y^​(i)。

从条件概率到交叉熵形式

现在，我们来看对于单个样本 iii 的条件概率 P(y(i)∣x(i))P(\mathbf{y}^{(i)} \mid \mathbf{x}^{(i)})P(y(i)∣x(i))。由于标签 y(i)\mathbf{y}^{(i)}y(i) 是独热编码向量（假设其真实类别为 j∗j^*j∗，即 yj∗(i)=1y_{j^*}^{(i)} = 1yj∗(i)​=1，其余 yj(i)=0,j≠j∗y_j^{(i)} = 0, j \neq j^*yj(i)​=0,j=j∗），观察到这个特定标签向量的概率，实际上就是模型预测其为真实类别的概率 y^j∗(i)\hat{y}_{j^*}^{(i)}y^​j∗(i)​。这是因为独热编码向量中，只有真实类别位置为1，其他都为0：

P(y(i)∣x(i))=∏j=1qP(y(i)=j∣x(i))yj(i)=∏j=1q(y^j(i))yj(i)P(\mathbf{y}^{(i)} \mid \mathbf{x}^{(i)}) = \prod_{j=1}^q P(y^{(i)}=j \mid \mathbf{x}^{(i)})^{y_j^{(i)}} = \prod_{j=1}^q (\hat{y}_j^{(i)})^{y_j^{(i)}} P(y(i)∣x(i))=j=1∏q​P(y(i)=j∣x(i))yj(i)​=j=1∏q​(y^​j(i)​)yj(i)​

这个等式的含义是：因为 yj(i)y_j^{(i)}yj(i)​ 只有在 j=j∗j=j^*j=j∗（真实类别）时为1，其他都为0，所以连乘的结果就是 y^j∗(i)\hat{y}_{j^*}^{(i)}y^​j∗(i)​。

于是，对于这个样本的负对数似然为：

−log⁡P(y(i)∣x(i))=−log⁡[∏j=1q(y^j(i))yj(i)]=−∑j=1qyj(i)log⁡y^j(i)-\log P(\mathbf{y}^{(i)} \mid \mathbf{x}^{(i)}) = -\log \left[ \prod_{j=1}^q (\hat{y}_j^{(i)})^{y_j^{(i)}} \right] = -\sum_{j=1}^q y_j^{(i)} \log \hat{y}_j^{(i)} −logP(y(i)∣x(i))=−log[j=1∏q​(y^​j(i)​)yj(i)​]=−j=1∑q​yj(i)​logy^​j(i)​

最后这一步运用了对数的性质 log⁡(ab)=blog⁡a\log(a^b) = b \log alog(ab)=bloga 以及连乘的对数等于对数的求和。

我们注意到，−∑j=1qyj(i)log⁡y^j(i)-\sum_{j=1}^q y_j^{(i)} \log \hat{y}_j^{(i)}−∑j=1q​yj(i)​logy^​j(i)​ 正是交叉熵损失函数（Cross-Entropy Loss） 的标准定义。它衡量了真实标签的分布 y(i)\mathbf{y}^{(i)}y(i)（这里是独热编码）与模型预测的分布 y^(i)\hat{\mathbf{y}}^{(i)}y^​(i) 之间的差异。
